## 声音设计（自定义描述）

用文字描述你想要的声音，例如：

```bash
# 低沉磁性的中年男声，像深夜电台主播
python3 generate_audio.py book.txt --voice-design "低沉磁性的中年男声，像深夜电台主播"

# 元气明亮的少年音
python3 generate_audio.py book.txt --voice-design "元气明亮的少年音" \
  --design-prompt "用充满朝气的语气朗读，语速轻快"
```

**支持描述维度：**
- 性别：男/女/中性
- 年龄：少年/青年/中年/老年
- 声线质感：磁性/沙哑/清亮/醇厚/柔和
- 情绪：温暖/沉稳/元气/神秘/严肃
- 语速：缓慢/正常/轻快
- 角色：旁白/主播/导师/讲故事

## 声音克隆

用 5 秒音频克隆任何声线：

```bash
python3 generate_audio.py book.txt --voice-clone --ref-audio 参考音频.mp3
```

**参考音频要求：**
- 时长：5-8 秒最佳
- 内容：清晰朗读，无背景音乐
- 格式：MP3/WAV/M4A
- 采样率：16kHz 以上

**克隆提示词（可选）：**
```bash
--clone-prompt "标准流利普通话"
--clone-prompt "温暖亲切"
--clone-prompt "沉稳磁性"
```
