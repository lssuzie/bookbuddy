# 🎯 声音匹配指南

> **新逻辑：先找到你喜欢的声音，然后用它读一切。**
> 不需要根据内容类型切换声音。同一个声音，调速度 + 调风格就够了。

---

## 核心原则：一个声音，不同演绎

```bash
# 找到你的声音
python3 generate_audio.py 书.txt --voice-design "温柔私语"

# 同一个声音，不同用法
python3 generate_audio.py 论文.md --voice-design "温柔私语"           # 1.0x 正常读
python3 generate_audio.py 故事.md --voice-design "温柔私语" -s 0.9    # 0.9x 慢一点，有感情
python3 generate_audio.py 播客.md --voice-design "温柔私语" -s 1.1    # 1.1x 快一点，像聊天
python3 generate_audio.py 睡前.md --voice-design "温柔私语" -s 0.7    # 0.7x 更慢，哄睡
```

## 找声音的三种方式

### 1. 试听内置音色

| 音色 | 语言 | 感觉 | 
|:-----|:----:|:-----|
| 冰糖 | 中文 | 清亮甜美 |
| 茉莉 | 中文 | 温柔知性 |
| 苏打 | 中文 | 清爽活力 |
| 白桦 | 中文 | 沉稳磁性 |
| Mia | 英文 | Bright |
| Chloe | 英文 | Warm |
| Milo | 英文 | Deep |
| Dean | 英文 | Calm |

### 2. 试听声音设计预设

| 预设 | 声线 | 
|:-----|:-----|
| **温柔私语** 👑 | 温暖沙哑，贴耳私语 |
| **知识讲述** | 沉稳清晰，像老师在娓娓道来 |
| **故事演绎** | 有戏剧张力，能演绎情绪 |
| **播客主持** | 自然亲切，像朋友在聊天 |
| **睡前陪伴** | 温柔缓慢，越来越轻 |
| **冥想引导** | 空灵安宁，像从远处传来 |

### 3. 自定义声音描述

```
[性别] + [年龄段] + [声线质感] + [风格]
```

```
"温暖甜美有气质的女声，像知性偶像在娓娓道来"
"低沉磁性有故事感的男声，沉稳而克制"
"年轻有活力的美式女声，语速轻快"
"元气明亮的少年音，20岁出头，充满阳光感"
```

## 声音克隆

```bash
python3 generate_audio.py 书.txt --voice-clone --ref-audio 参考音频.mp3
```

**参考音频要求：** 5-8 秒，清晰朗读，无背景音乐

**克隆后也可以调速度和风格：**
```bash
python3 generate_audio.py 论文.md --voice-clone --ref-audio 我的声音.mp3           # 1.0x
python3 generate_audio.py 睡前.md --voice-clone --ref-audio 我的声音.mp3 -s 0.7    # 0.7x
```
