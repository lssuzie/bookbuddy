## 声音设计描述模板

### 基本公式

```
[性别] + [年龄段] + [声线质感] + [角色/场景] + [情绪]
```

### 示例库

```
"低沉磁性的中年男声，像深夜电台主播，温暖而克制"
"元气明亮的少年音，20岁出头，充满活力和阳光感"
"温柔知性的女声，30岁左右，像是在给好朋友讲故事"
"空灵安宁的女声，像从远处传来的冥想引导"
"沉稳厚重的老年男声，像一位智慧的长者在传授经验"
"清亮甜美的少女音，活泼可爱，语速轻快"
"冷静克制的新闻主播声线，字正腔圆，不带感情色彩"
"沙哑有质感的烟嗓男声，像经历了沧桑的说书人"
```

### 预设（内置在 generate_audio.py 中）

| 预设名 | 效果 | 适合 |
|:------|:----|:----|
| `睡前催眠` | 温和缓慢，越来越轻 | 睡前读物、散文 |
| `冥想引导` | 空灵安宁，有节奏的停顿 | 正念、冥想类内容 |
| `深夜电台` | 磁性醇厚，亲切不亲密 | 散文、随笔、情感类 |
| `温柔叙述` | 柔和温暖，像姐姐讲故事 | 童话、育儿、轻小说 |
| `Her知性元气` | 略带沙哑的温暖女声，充满生命力和陪伴感 | 一切——这是BookBuddy的默认灵魂声线 |
| `悬疑小说` | 沉稳男中音，神秘悬念感 | 悬疑/推理/惊悚小说 |
