# Skill 声音克隆指南

## 什么是声音克隆

用 5 秒参考音频，克隆任何声线。Mimo TTS 的 `mimo-v2.5-tts-voiceclone` 模型支持零样本语音克隆。

## 参考音频要求

| 要求 | 说明 |
|:-----|:-----|
| **时长** | 5-8 秒最佳（太短不够特征，太长浪费） |
| **内容** | 清晰朗读，无背景音乐，无环境噪音 |
| **格式** | MP3/WAV/M4A 均可 |
| **采样率** | 16kHz 以上 |
| **语言** | 中文参考音频 → 中文输出；英文参考音频 → 英文输出 |

## 录制建议

1. **安静环境**：关窗、关风扇、远离电脑风扇
2. **手机录音**：用语音备忘录，靠近嘴巴（10-20cm）
3. **内容**：读一段 5-8 秒的文字，例如：
   > "你好，我是 [你的名字]。今天天气很好，阳光明媚。"
4. **检查**：回放确认无杂音、无爆音

## 使用方法

```bash
# 基本克隆
python3 generate_audio.py 书.txt --voice-clone --ref-audio 我的声音.mp3

# 指定克隆风格
python3 generate_audio.py 书.txt --voice-clone --ref-audio 参考.mp3 \
  --clone-prompt "标准流利普通话"

python3 generate_audio.py 书.txt --voice-clone --ref-audio 参考.mp3 \
  --clone-prompt "温暖亲切"

python3 generate_audio.py 书.txt --voice-clone --ref-audio 参考.mp3 \
  --clone-prompt "沉稳磁性"
```

## 克隆提示词（可选）

| 提示词 | 效果 |
|:-------|:-----|
| `标准流利` | 标准普通话，无口音 |
| `温暖亲切` | 声音更柔和，有亲和力 |
| `沉稳磁性` | 声音更低沉，更有质感 |
| `元气明亮` | 声音更轻快，有活力 |
| `知识讲述` | 适合讲故事，语速适中 |

## 伦理提醒

> ⚠️ **请只使用你自己的声音，或你已获授权的声音参考。**

- 克隆自己的声音：✅ 安全
- 克隆家人的声音（经同意）：✅ 安全
- 克隆公众人物/明星的声音：⚠️ 法律风险
- 克隆他人声音用于冒充：❌ 禁止

## 常见问题

| 问题 | 原因 | 解决方案 |
|:-----|:-----|:---------|
| 克隆声音不像 | 参考音频质量差 | 重新录制，确保清晰无噪 |
| 克隆声音有杂音 | 参考音频有背景音 | 用降噪工具处理参考音频 |
| 克隆声音读错字 | 参考音频语言不匹配 | 确保参考音频语言与目标文本一致 |
| 克隆失败 | API 限制 | 确认 API Key 有效，参考音频格式正确 |

## 进阶：多段参考

如果有一段长音频（如播客），可以截取其中 5-8 秒清晰片段作为参考：

```bash
# 用 ffmpeg 截取 5 秒
ffmpeg -i 长音频.mp3 -ss 00:01:23 -t 5 -ac 1 参考音频.mp3

# 然后用截取片段克隆
python3 generate_audio.py 书.txt --voice-clone --ref-audio 参考音频.mp3
```

## 与声音设计的区别

| 特性 | 声音克隆 | 声音设计 |
|:-----|:---------|:---------|
| **输入** | 5 秒音频 | 文字描述 |
| **输出** | 克隆参考音频的声音 | 按描述创造新声音 |
| **相似度** | 高（与参考音频相似） | 中（按描述近似） |
| **适合** | 想用自己的声音/特定人声 | 想设计独特声线 |
| **API 模型** | `mimo-v2.5-tts-voiceclone` | `mimo-v2.5-tts-voicedesign` |
